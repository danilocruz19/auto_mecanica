# Auto Mecânica 🚗  

Este é um app Flutter para gerenciamento de serviços automotivos.

## 📲 Funcionalidades
- 📌 Cadastro de veículos
- ⚙️ Gerenciamento de serviços
- 📅 Agendamento de manutenções

## 🚀 Tecnologias
- Flutter 3.0
- Provider (gerenciamento de estado)
